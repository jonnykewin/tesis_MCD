# DETERMINACIÓN DE ISLAS DE CALOR EN BARRANQUILLA, CARTAGENA Y SANTA MARTA

Resumen
-------
Repositorio con el código, notebooks, datos procesados, modelos y resultados del proyecto de grado "DETERMINACIÓN DE ISLAS DE CALOR EN LAS CIUDADES DE BARRANQUILLA, CARTAGENA Y SANTA MARTA A PARTIR DE IMÁGENES SATELITALES Y ALGORITMOS DE MACHINE LEARNING", realizado por Jonny Carlos Sanches y Cristian Camilo Rodriguez, Maestría en Ciencia de Datos — Pontificia Universidad Javeriana.

Propósito
---------
Desarrollar y documentar un flujo reproducible para identificar y caracterizar islas de calor urbano (UHI) en tres ciudades del Caribe colombiano usando imágenes satelitales y técnicas de machine learning, y producir mapas y métricas útiles para análisis urbano.

Contenido principal
-------------------
- Notebooks: preprocesamiento, análisis exploratorio, entrenamiento, evaluación y generación de mapas.
- Código: scripts para procesamiento geoespacial y modelado.
- Datos procesados: capas y derivados listos para uso en los experimentos.
- Modelos y resultados: pesos entrenados, métricas y visualizaciones finales.

Consideraciones
---------------
- Los notebooks incluyen instrucciones para regenerar los datos desde las fuentes originales cuando aplique.
- Algunas decisiones metodológicas (resolución, umbrales, ventanas temporales) están documentadas en las notas metodológicas y pueden ajustarse según el contexto.
